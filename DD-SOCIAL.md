## S1: Language

### Claim: Symbolic communication is forced

**Required inputs:**
1. Social organisms exist (B5)
2. Coordination increases fitness
3. Neural systems can learn (B6)

**Derivation:**
1. Cooperation requires coordination
2. Coordination requires information transfer
3. Direct perception limited (distance, time)
4. Symbols extend perception beyond immediate
5. Symbolic communication is attractor

**Status:** FORCED (existence)

**DD interpretation:** Language = externalized distinguishability. Symbols = portable distinctions.

---

### Claim: Language must be compositional

**Analysis:**
1. Finite memory, infinite situations
2. Non-compositional: one symbol per situation → memory overflow
3. Compositional: combine symbols → infinite situations from finite set
4. Criticality selects compositional (sustainable Φ)

**Status:** FORCED

**DD interpretation:** Compositionality = tensor structure of meaning. Sentences = factorized distinctions.

---

### Claim: Grammar is forced (not arbitrary)

**Analysis:**
1. Compositional language needs combination rules
2. Arbitrary combinations → ambiguity → Φ → 0 (no information)
3. Structured rules → disambiguation → stable Φ
4. Grammar = constraints that preserve distinguishability

**Status:** FORCED (existence of structure) / BOUND (specific rules)

**DD interpretation:** Grammar = criticality constraints on symbol combination.

---

### Claim: Universal Grammar features exist

**Analysis:**
If grammar is forced by criticality, certain features should be universal:
1. Recursion (finite rules, infinite expressions)
2. Categories (noun/verb distinction)
3. Reference (symbols point to distinctions)

**Status:** FORCED (universal features) / BOUND (specific implementations)

**Empirical check:** Chomsky's universals — DD predicts they exist for structural reasons, not biological accident.

4. Law = formalized norm system

**Status:** FORCED (given scale)

**DD interpretation:** Law = explicit encoding of distinguishability constraints for large groups.

3. Direct democracy has O(N²) communication cost

1. Flat structure: everyone coordinates with everyone → O(N²)

3. Criticality selects lower complexity
4. Hierarchy emerges at scale

**Status:** FORCED (at large N)

**DD interpretation:** Hierarchy = compression of coordination distinguishability. Leaders = distinction aggregators.

**Analysis:**

3. Critical regime: distributed enough for resilience, concentrated enough for action

**Status:** FORCED (bounds exist) / BOUND (specific values)

**DD interpretation:** Optimal power distribution = critical balance. Tyranny and anarchy both violate criticality.
